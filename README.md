
# VietHoa 9Tech

Dự án giúp việt hoá game tự động trên nintendo switch. 

Warning: Only support in Vietnamese



## Installation


```bash
  git clone https://github.com/PhamMinhKha/VietHoa-N
  cd VietHoa-N
  npm install 
    
```
   
## Developer

To Developer this project run

```bash
  npm run dev
```
